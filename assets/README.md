Welcome to Pixel Desk! This is a website where only select users can access the webspace. Share the password with your friends and you can send notes to eachother, listen to good music, and explore fun recipes together. 
Password: 36

## Project Overview
The application allows users to:
- create and manage notes
- listen to music
- explore recipes
- interact with a desktop interface

## How to Open the Website
1. Copy and paste the link to your browser: https://redp1337.github.io/pixel-desk/
2. Enter the password
3. Explore the desktop.
4. Enjoy!

## Dependencies
This project uses:
- HTML
- CSS
- Javascript

## Features
feature/login
- login page with interactive password entry
- only users with the correct password can access the Pixel Desk desktop

feature/desktop
- desktop interface with icons

feature/taskbar-clock 
- includes a functional clock in military time

feature/notepad
- user can create notes with a title and content

feature/notes-folder
- saved notes will appear here
- user can delete them if they are no longer needed

feature/music-player
- interactive controls that allow the user to play, pause, skip, and select the previous song

feature/recipe-book
- user can view fun and tasty fruit recipes here

feature/open-close-windows
- this opens and closes application windows depending on which icon is clicked

## Team Reflection
Throughout this project, the team learned how to collaborate using GitHub, manage feature branches, and resolve merge conflicts. One challenge we faced was integrating all features together without breaking functionality/look of the desktop. We solved this by testing frequently and fixing bugs after merging. We also improved our communication and planning skills by dividing tasks clearly and supporting each other when issues came up. Overall, this project helped us better understand GitHub workflows and practice using HTML, CSS, and Javascript.
